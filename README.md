# Vgmify 🎮🎵🌟

> **A Video Game Music Radio Station** playing **24/7** on Roku!

## Features 🚀
- Endless loops of classic and modern video game tracks 🎶
- Easy navigation on Roku 📺
- Frequent updates with fresh soundtracks 🔄

## Installation ⚙️
1. Clone or download this repo 📂
2. Follow Roku channel setup instructions 🏗️
3. Enjoy endless VGM! 🎉
